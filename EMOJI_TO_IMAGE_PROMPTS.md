# Project Infinity Idle - Prompts para substituir emojis por imagens (grades 3x3)

**189 icones de conteudo** organizados em **21 grades 3x3**. Cada bloco abaixo e um prompt pronto para colar no ChatGPT (gerador de imagem). Gere uma imagem por grade, depois recorte as 9 celulas.

> Glifos de interface (setas, X, check, menu, som, salvar/exportar) foram deixados de fora a pedido - devem continuar como fonte/CSS.
